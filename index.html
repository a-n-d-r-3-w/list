List

A list app implemented using MongoDB, Express, Angular, and Node.js, and tested using WebdriverIO and Mocha

View on GitHub Download .zip Download .tar.gz

List is a web application that allows the user to create, edit, delete, and move list items, as well as mark them complete. The UI is designed to be minimal, with elements appearing only when necessary.

You can try it here. (It might take a minute to load the first time because it's running on a free dyno on Heroku.)
